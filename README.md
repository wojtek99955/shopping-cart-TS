# Fancy Shop

## About

Store App

## Live

Link : https://wojtekk-dev-fancy-shop.netlify.app/

## Technologies

- reactjs
- typescript
- formik
- context api
- styled-components

## Features

- add items to the cart
- sort items
- add to favorites
- checkout proccess

## Installation

Download or clone the repository

````bash
  npm install
  npm start
```
````
